```

```

### Installation

**1. Clone repository**
```
git clone https://github.com/yourusername/scamshield.git
cd scamshield
```

**2. Install Langflow**
```
pip install langflow
```

**3. Start Langflow**
```
langflow run
```

**4. Add API key inside Langflow**

